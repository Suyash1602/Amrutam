# Amrutam - Grow Your Practice

Amrutam is a comprehensive platform designed for Ayurvedic practitioners to digitize their practice, connect with a broader patient base, and manage their professional workflow efficiently. Built with **React 19** and **Vite**, the application offers a responsive and modern interface for healthcare management.

## Features

### 1. Doctor Onboarding

A simple, four-step process to join the Amrutam specialist network:

- **Step 1**: Contact Amrutam to receive a unique referral code.
- **Step 2**: Register through the Amrutam Doctors app or official website.
- **Step 3**: Complete KYC by uploading required verification documents.
- **Step 4**: Start providing consultations once verified.

### 2. Flexible Consultation Modes

Practitioners can choose how they interact with patients to suit their preferences:

- **Instant Chat**: For quick text-based queries.
- **Instant Call**: For immediate voice consultations.
- **Scheduled Video Calls**: For in-depth personalized care.

### 3. Practice Management Tools

- **Smart Wallet**: Securely manage earnings and request withdrawals with OTP-protected transactions.
- **Flexible Timing**: Use the availability toggle to control when patients can book consultations, ensuring a balanced practice.
- **Patient Insights**: Access essential patient details such as age, gender, height, weight, and specific concerns (e.g., immunity) before a session.

### 4. Professional Recognition

- **Featured Publications**: Amrutam has been recognized by leading publications including Forbes, Vogue, GQ, and The Times of India.
- **Expert Endorsements**: Trusted by BAMS and Ayurvedic practitioners who recommend Amrutam for its commitment to authentic, chemical-free, and holistic well-being.

## Technical Stack

- **Frontend Library**: React 19.2.4
- **Build Tool**: Vite 8.0.1
- **Styling**: Tailwind CSS 4.2.2 with PostCSS
- **Fonts**: Poppins and DM Sans
- **Linting**: ESLint 9.39.4

## Project Structure

- `src/components/`: Modular UI components such as `Navbar`, `Hero`, `Wallet`, and `JoinSteps`.
- `src/pages/`: Main page layouts (e.g., `Home.jsx`).
- `src/assets/`: Brand logos and high-quality imagery for the doctor-facing interface.
- `tailwind.config.js`: Custom theme configuration including the primary brand color (`#3A643C`) and specific font families.

## Getting Started

### Installation

1.  Clone the repository.
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Start the development server:
    ```bash
    npm run dev
    ```

### Available Scripts

- `npm run dev`: Runs the app in development mode.
- `npm run build`: Builds the app for production.
- `npm run lint`: Checks the code for linting errors.
- `npm run preview`: Previews the production build locally.
